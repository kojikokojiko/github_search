# Features
* Riverpod+MVMM+Repositoryで簡潔なアーキテクチャを意識。
* Api_Client ,Repositoryに対して、testを実装。
* 画面遷移時のアニメーションをHeroで実装。
* レポジトリ名をAnimatedTextでタイピングテキスト風に表示。
